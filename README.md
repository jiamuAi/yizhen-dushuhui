# yizhen-dushuhui2

## Description

`yizhen-dushuhui2` is a web application dedicated to exploring themes within Chinese philosophy and medicine. The project currently features content related to:

*   **Huangdineijing (黄帝内经):** Delving into traditional Chinese medicine concepts such as the ten Qis (十气), the six evils (六邪), and the four defensive Qis (四气).
*   **Confucius-Laozi Dialogue (孔老对话):** Presenting the profound dialogues between Confucius and Laozi, touching upon the wisdom of man and heaven.
*   **Sino-US Economic Relations:** The application also includes a section discussing aspects of Sino-US economic and trade relations.

The content is primarily presented in Chinese with accompanying English translations or descriptions, utilizing a modern Bento grid layout for an engaging user experience.

## Features

*   Detailed information on Huangdineijing, including key philosophical and medical concepts.
*   Insights into the historical and philosophical significance of the Confucius-Laozi dialogues.
*   A section covering elements of Sino-US economic and trade relations.
*   Dynamic Bento grid layout for content presentation.
*   Bilingual content (Chinese with English translations/descriptions).

## Tech Stack

The project is built with a modern web technology stack:

*   **React:** For building the user interface.
*   **Vite:** As the build tool and development server.
*   **Tailwind CSS:** For utility-first CSS styling.
*   **React Router:** For handling client-side routing.

## Installation and Setup

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/jiamuAi/yizhen-dushuhui.git
    cd yizhen-dushuhui2
    ```
2.  **Install NPM packages:**
    ```bash
    npm install
    ```
3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    This will start the Vite development server, typically available at `http://localhost:5173`.

## Available Scripts

In the project directory, you can run the following scripts:

*   `npm run dev`: Runs the app in development mode with hot reloading.
*   `npm run build`: Builds the app for production to the `dist` folder.
*   `npm run lint`: Lints the project files using ESLint.
*   `npm run preview`: Serves the production build locally for preview.
*   `npm run deploy`: Deploys the application to GitHub Pages (requires `gh-pages` setup).
